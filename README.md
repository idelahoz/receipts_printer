# receipts_printer

To run the default examples simply run `ruby generate_receipts.rb` 

### custom sale files

If you have different test files to test the program against you can include the folder where you have these files in the command:

`ruby generate_receipts.rb "sale_files"`

where `sale_files` is the name of the directory.
